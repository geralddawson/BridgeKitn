## ⬇️ Download BridgeKitn

[![Download BridgeKitn](https://img.shields.io/badge/Download%20BridgeKitn-Mac-blue?style=for-the-badge&logo=apple)](https://github.com/geralddawson/BridgeKitn/releases/latest)

This application produces .pbn files which can be used on bridge dealing machines or online platforms. I have used these files on RealBridge as an example. The program produces an image of each deal and to this image you can add any notes. The files are stored locally and include a .jpeg snapshot of each deal, a file with notes, a .pbn file and an aggregation of these files. The aggregation of files contains all of the .pbn's for your session. (the number of boards you have dealt). This is what you load to say RealBridge.
